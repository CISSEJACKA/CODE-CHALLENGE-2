
# Concerts Domain Code Challenge

This repository contains a Python project for managing a concerts domain, including bands, venues, and concerts. The project demonstrates the use of Object-Oriented Programming (OOP) concepts, object relationships, and aggregate and association methods.

## Project Overview
The project involves three main classes:
 A Band has many `Concerts.
- A venue has many Concerts`.
- A Concert belongs to a Band and a Venue`.
- The relationship between Band and Venue is many-to-many through `Concert.
